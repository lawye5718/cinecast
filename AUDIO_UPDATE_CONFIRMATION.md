## 🚀 使用说明

### 过渡音效使用
系统默认使用新的哲理过渡音效，在章节切换时自动播放。

### 环境音切换
可以通过配置文件切换环境音主题：
```json
{
    "ambient_theme": "fountain"  // 使用喷泉环境音
    // 或 "iceland_wind"       // 使用默认冰岛风声
}
```

### 音频规格标准
所有音频文件都会自动转换为统一规格：
- 采样率: 22050Hz
- 声道数: 1 (单声道)
- 格式: WAV (内部处理)
